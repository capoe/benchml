class benchml.pipeline.TransformBase(**kwargs)

Bases: object

Transforms constitute the nodes of a pipeline. Each transforms implements at least one of three methods, through which it acts on the data stream: map, fit, and feed.

Parameters:

  • tag (str, optional) – Address of the transform via which its outputs can be referenced. Default: self.__class__.__name__

  • args (dict, optional) – Dictionary of transform parameters. Default: {}

  • inputs (dict, optional) – Dictionary specifying input streams Default: {}
